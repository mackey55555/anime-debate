# 🎬 プレゼンスライド（reveal.js）

「アニメ考察ひろば」の発表用スライド。夏休み × アニメの雰囲気＋アニメーション多め。
**中身はプレースホルダ**（`〔 〕` や `差し替え` の箇所）です。あとで実内容に差し替えてください。

## 起動方法

いちばん簡単なのは、`index.html` をブラウザで直接ひらくだけ：

```bash
open slides/index.html      # Mac
```

うまく表示されない/フォントが出ない場合は、簡易サーバー経由がおすすめ：

```bash
cd slides
npx serve .                 # → 表示されたURLを開く
# または
python3 -m http.server 8080 # → http://localhost:8080
```

> reveal.js・フォントは CDN から読み込みます。**発表当日はネット接続があると安心**です。
> オフライン確実にしたい場合は reveal.js とフォントをローカルに落として差し替えます（必要なら対応します）。

## 操作

| キー | 動作 |
|---|---|
| → / ← / Space | スライド移動 |
| F | フルスクリーン |
| S | スピーカーノート（発表者ビュー） |
| O | 一覧（オーバービュー） |
| B | 画面を暗く |

## 構成（スライド順）

1. タイトル
2. コンセプト（夏のイラストは自作SVG）
3. 課題（ストーリー仕立て・複数枚／イラスト自作SVG）
4. 先生／生徒それぞれの機能
5. ユーザーフロー
6. ★ ライブデモ枠 ★（実機を動かす）
7. 技術構成（アーキテクチャ図は自作SVG）
8. 今後の拡張案
9. クロージング

> 画像は差し込み枠ではなく **SVGで作図済み**（コンセプト絵・課題イラスト・構成図）。デモは実機で見せる想定のため画面キャプチャ枠は無し。

## 差し替え方

- テキスト: `〔 〕` で囲った部分を書き換え。`<span class="todo">差し替え</span>` は消してOK
- 画像: `<div class="imgslot">…</div>` を `<img src="画像パス" />` に置き換え
  - 画像は `slides/images/` などに置くと管理しやすいです
- スライドの増減: `<section>…</section>` 単位でコピー／削除
- 見た目の色・アニメは `theme.css` で調整

## ファイル

- `index.html` … スライド本体（内容）
- `theme.css` … 夏×アニメのカスタムテーマ＋アニメーション
